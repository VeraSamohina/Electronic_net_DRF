# Онлайн платформа торговой сети электроники

Этот проект представляет собой веб-приложение, предназначенное для управления иерархической структурой сети по продаже электроники. 

Сеть представляет собой иерархическую структуру из 3 уровней:

- Завод;
- Розничная сеть;
- Индивидуальный предприниматель.

Каждое звено сети ссылается только на одного поставщика оборудования. Уровень иерархии определяется не названием звена, а отношением к остальным элементам сети,
т.е. завод всегда находится на 0 уровне, а если розничная сеть относится напрямую к заводу, минуя остальные звенья - её уровень - 1.

## Инструкция по Запуску
Для запуска проекта выполните следующие шаги:

- Клонирование репозитория:
`git clone git@github.com:VeraSamohina/Electronic_net_DRF.git`
`cd electronic`

- Установка зависимостей:
`pip install -r requirements.txt`

- Создайте файл .env в корневой директории проекта и добавьте в него переменные среды в соответсвии с примером (.env.example)

- Создайте базу данных для проекта через psql или pgAdmin.

- Применение миграций:
`python manage.py migrate`

- Запуск сервера:
`python manage.py runserver`

Доступ к API только у активных аутетифицированных пользователей. 
Созданные объекты отображаются в админ-панели.

На странице объекта сети реализована:
- ссылка на «Поставщика»;
- фильтр по названию города;
- «admin action», очищающий задолженность перед поставщиком у выбранных объектов.
